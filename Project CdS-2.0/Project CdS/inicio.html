Project CdS

O Projeto Caixinha de Sonhos (CdS) tem como objetivo restaurar sonhos — ou pesadelos — coletivos, experiências oníricas tão específicas e recorrentes que despertaram a atenção de nossa equipe. Estamos reunindo relatos de diferentes pessoas que descrevem sonhos ou visões semelhantes, às vezes idênticas, em busca de padrões e significados ocultos. Para isso, utilizamos o Somnovisor, um dispositivo capaz de decodificar sonhos a partir de ondas cerebrais, projetando-os em cartões de memória. Esses registros são então processados pelo Filtro dos Sonhos, onde são filtrados e armazenados para análise. O Somnovisor permite tanto salvar quanto explorar sonhos, oferecendo uma janela para os mistérios mais profundos da mente humana. Sinta-se à vontade para mergulhar nos enigmas da CdS — mas lembre-se: nada do que for visto aqui ultrapassará os limites de um sonho para aqueles que ousarem visitar nosso projeto.
